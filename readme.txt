Schedule updates to page or post content automatically and on a schedule, compatible with Elementor.

== Description ==
Automatic Scheluded Content, allows you to schedule updates to the content, automatically. 
It's very simple, you just clone, make the changes and schedule a publishing time. It is compatible with Elementor.

== Installation ==
1. Install and activate.
1. The plugin adds an additional option to clone the pages.
1. Clone the desired content, select a publishing time.
